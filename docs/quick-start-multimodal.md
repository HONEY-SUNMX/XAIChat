# 多模态聊天快速开始

> 5 分钟快速体验 XAIChat 的多模态聊天功能！

---

## 🚀 快速开始（3 步）

### 步骤 1: 启动服务

```bash
# 进入项目目录
cd XAIChat

# 启动完整 Web 应用（前后端）
./start_web.sh

# 或者分别启动
./start_web.sh backend   # 后端 API（端口 8000）
./start_web.sh frontend  # 前端界面（端口 5173）
```

等待服务启动完成（约 10-30 秒）。

### 步骤 2: 打开浏览器

访问 http://localhost:5173

默认会打开**多模态聊天**页面（左侧有紫色 "NEW" 标签）。

### 步骤 3: 开始使用

#### 场景 A: 文本对话 💬
直接输入文字，按 Enter 发送：
```
你: 你好，介绍一下你自己
```

#### 场景 B: 图像理解 🖼️
1. 点击输入框右侧的 📷 图标
2. 选择一张图片
3. 输入问题："这是什么？"
4. 发送

#### 场景 C: 图片生成 🎨
输入包含生成关键词的文字：
```
你: 画一只可爱的猫咪坐在草地上
```

系统会自动检测并生成图片！

## 📖 详细使用技巧

### 💡 提示词技巧

#### 触发图片生成的关键词

**中文**:
- "画..."
- "生成图片..."
- "帮我画..."
- "请画..."
- "创作..."

**英文**:
- "draw..."
- "generate image of..."
- "create image of..."

**示例**:
```
✅ "画一只可爱的猫咪"         → 会生成图片
✅ "帮我画一个红色的苹果"     → 会生成图片
✅ "generate a beautiful sunset" → 会生成图片
❌ "猫咪是什么样子的？"       → 普通对话
```

#### 图片生成最佳实践

**好的提示词**:
```
✅ 具体："画一只橘色的猫咪，坐在绿色的草地上，阳光明媚"
✅ 详细："一只可爱的金毛犬，奔跑在海滩上，背景是蓝天白云"
```

**不好的提示词**:
```
❌ 太简单："画只猫"
❌ 太模糊："画个东西"
```

## ❓ 常见问题

### Q1: 首次使用需要等待多久？
**A**: 首次使用需要下载模型（约 1-4GB），取决于网络速度，通常 5-30 分钟。后续使用即开即用。

### Q2: 生成一张图片需要多长时间？
**A**:
- **CPU 模式**: 10-30 秒（默认）
- **GPU 模式**: 2-5 秒（需要 NVIDIA GPU）

### Q3: 支持哪些图片格式？
**A**: JPEG、PNG、GIF、WebP 等常见格式。

### Q4: 图片大小有限制吗？
**A**:
- **上传**: 建议 < 5MB
- **生成**: 默认 512x512，可调整（256-1024）

### Q5: 可以生成什么样的图片？
**A**:
- ✅ 风景、动物、物品、人物
- ✅ 艺术风格、抽象图案
- ❌ 不支持生成人脸特写（隐私保护）
- ❌ 不支持暴力、色情等内容

### Q6: 思考模式有什么用？
**A**: 展示 AI 的推理过程，帮助：
- 理解 AI 如何得出答案
- 发现潜在的理解偏差
- 学习 AI 的思维方式

### Q7: 会话历史保存在哪里？
**A**:
- 浏览器本地存储（localStorage）
- 重启浏览器后会自动恢复
- 清除浏览器数据会丢失历史

### Q8: 如何保存生成的图片？
**A**:
- 右键图片 → 另存为
- 或在 `outputs/` 目录查找（文件名显示在图片下方）

## 🎓 进阶使用

### API 调用

如果你想在代码中使用多模态功能：

```bash
# 文本对话
curl -X POST http://localhost:8000/api/multimodal/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "你好", "stream": false}'

# 图像理解
curl -X POST http://localhost:8000/api/multimodal/chat-with-image \
  -F "message=这是什么？" \
  -F "file=@./image.jpg"

# 图片生成（自动触发）
curl -X POST http://localhost:8000/api/multimodal/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "画一只猫", "stream": false}'
```

### 查看 API 文档

访问 http://localhost:8000/docs 查看完整的交互式 API 文档。
